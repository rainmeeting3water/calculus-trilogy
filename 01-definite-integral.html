三、定積分概念的 5 個步驟Five Steps to the Definite Integral

Step 1：從一個矩形開始Start with one rectangle

矩形面積 = 底 × 高。

在函數圖形下，一個小矩形的底長是 |Δx|，高度大小大約是 |f(xi*)|。

但定積分保留方向與正負號，所以第 i 塊對定積分的有向貢獻是 f(xi*)Δx。

公式：第 i 塊有向貢獻 = f(xi*)Δx

Step 2：把區間 [a, b] 切成很多小段Subdivide the interval [a, b]

如果把從 a 到 b 的距離切成 n 等分，每一小段的有向寬度是：

Δx = (b - a) / n

若 a < b，Δx 為正；若 a > b，Δx 為負，表示累積方向相反。

Step 3：把很多小矩形加起來Add the small rectangles

把所有小矩形的有向貢獻加起來（稱為黎曼和），就可以得到近似的總累積量：

n Σ i=1 f(xi*) Δx

（其中 xi* 可以選起點端點、終點端點或中點。若 a < b，起點端點就是左端點；若 a > b，起點端點會在右側，表示累積方向已經反過來。）

Step 4：讓切割越來越細Make the subdivision finer

當 n 越來越大，每段長度 |Δx| 越來越小，小矩形的頂部就會越貼近函數曲線。

當矩形總和越來越穩定，趨近於一個極限值時，我們就把這個極限值記為定積分符號：

∫ab f(x) dx

Step 5：理解上下限與方向Interpret limits and direction

a 是開始累積的位置，b 是結束累積的位置。

  • 若 a = b，沒有寬度，所以：∫aa f(x) dx = 0
  • 若 a > b，代表累積的方向反過來（Δx 變為負），所以定積分的值會變號：
    ∫ab f(x) dx = - ∫ba f(x) dx

四、定積分是「帶符號的面積」A Definite Integral as Signed Area

定積分不只是普通的幾何面積，而是帶符號的累積量。

  • 高度 f(x) > 0 (圖形在 x 軸上方)：在向右累積 (a < b) 時貢獻正值；若向左累積 (a > b) 則貢獻負值。
  • 高度 f(x) < 0 (圖形在 x 軸下方)：在向右累積 (a < b) 時貢獻負值；若向左累積 (a > b) 則貢獻正值。
  • 所以定積分算出來的結果可能是正的、負的，也可能是 0。
  • 若正面積和負面積剛好互相抵消，定積分可能等於 0，但這不代表該區間內「沒有圖形面積」。

在一般從左到右累積，也就是 a < b 的情況下，x 軸上方貢獻正值，x 軸下方貢獻負值；若 a > b，累積方向反過來，定積分值會變號 / Reversing direction changes the sign of the definite integral。

五、定積分的基本性質Basic Properties of Definite Integrals

1. 上下限相同，累積量為 0

∫aa f(x) dx = 0

說明：從 a 累積到 a，沒有寬度，所以累積量為 0。

2. 上下限交換，符號改變

∫ab f(x) dx = - ∫ba f(x) dx

說明：累積方向反過來，總量會變號。

3. 區間可加性

∫ab f(x) dx = ∫ac f(x) dx + ∫cb f(x) dx

說明：從 a 到 b 的累積量，可以拆成從 a 到 c 加上從 c 到 b。

這個性質之後會幫助我們理解：
A(x+Δx) - A(x) = ∫xx+Δx f(t) dt

4. 常數倍可以提出來

∫ab k f(x) dx = k ∫ab f(x) dx

說明：如果每一點的高度都變成 k 倍，總累積量也會變成 k倍。

5. 和可以拆開

∫ab [f(x) + g(x)] dx = ∫ab f(x) dx + ∫ab g(x) dx

說明：兩個函數相加後的累積量，可以分開累積再相加。

六、下一步：讓上限動起來Next: Let the Upper Limit Move

這也是通往微積分基本定理的橋樑。

如果下限 a 固定，但上限不再是固定的數字 b，而是會移動的位置 x，我們可以定義一個新的面積函數：

A(x) = ∫ax f(t) dt

這裡用 t 當積分變數，是為了避免和上限 x 混淆。

這個 A(x) 代表從 a 累積到 x 的總量。

思考問題（下個模組的重點）：

A(x+Δx) - A(x) = ∫xx+Δx f(t) dt

✨ 總結 ✨Summary

定積分的核心不是先背公式，而是理解：

  1. 把區間切成很多小段。
  2. 每段用小矩形近似。
  3. 把每塊小矩形的有向貢獻加起來。
  4. 當切割越來越細，總和趨近一個穩定值。
  5. 這個穩定值就是定積分。
b ∫ a f(x) dx = lim n→∞ n Σ i=1 f(xi*) Δx
