不定積分：從微分倒著走Indefinite Integrals: Anti-differentiation and Families of Curves

理解 ∫ f(x)dx = F(x) + C 的真正意義Interpret an indefinite integral as a family of anti-derivatives.

我們先從熟悉的微分開始。
如果某個函數微分後會變成 f(x)，那麼我們能不能反過來，找出微分前的函數？If a function has derivative f(x), can we work backward to find an anti-derivative?

一、核心概念：兩種運算的差別Core Idea: Differentiation and Anti-differentiation

▶ 微分：已知 F(x)，求它的導函數 F'(x)Differentiation: find the derivative F'(x)

已知一個函數 F(x)，求它的變化率 F'(x)。Given a function F(x), differentiation finds its rate of change F'(x).

▶ 不定積分：已知變化率，反推原函數Indefinite integration: work backward from a rate of change

已知 f(x)，反過來找一個函數 F(x)，使得 F'(x) = f(x)。這樣的 F(x) 稱為反導函數，也常稱為原函數anti-derivative / antiderivative。Given f(x), indefinite integration works backward to find a function F(x) such that F'(x)=f(x). This F(x) is an anti-derivative.

💡 注意：不定積分不是單一答案，而是一整族函數 F(x) + Ca family of curves，其中 C 是任意常數。Note: an indefinite integral is not a single answer. It represents a family of curves F(x)+C, where C is an arbitrary constant.

三、為什麼會有一整族函數？（5 個概念步驟）Why Is There a Family of Curves?

Step 1：從微分開始Start with differentiation

我們知道：d/dx (x2) = 2xWe know this differentiation fact.

Step 2：反過來問Ask the reverse question

如果已知結果是 2x，誰微分後會得到 2x？If the derivative is 2x, which function could have produced it?

Step 3：發現不只一個答案Notice there is more than one answer

x2、x2 + 3、x2 - 5 微分後都會得到 2x。All of these functions have derivative 2x.

因為在圖形上，加減常數只是把曲線上下平移，每一點的切線 gradient (slope)（rate of change）完全沒有改變。Adding or subtracting a constant only shifts the curve vertically; the gradient and rate of change at each point do not change.

Step 4：常數 C 的意義Meaning of the constant C

因為常數微分後會變成 0，所以當我們反向尋找時，原本的常數是多少就「遺失」了，無法唯一決定。A constant differentiates to 0, so when we work backward, the original constant cannot be uniquely determined.

因此我們用 C 來代表所有可能的常數：Therefore we use C to represent all possible constants.

∫ 2x dx = x2 + C

Step 5：一般化公式General form

若找到一個函數 F(x) 滿足 F'(x) = f(x)，則：If we find a function F(x) satisfying F'(x)=f(x), then the general indefinite integral is as follows.

∫ f(x) dx = F(x) + C

這是微積分中最重要的連結之一。This is one of the most important connections in calculus.

在這個模組中，我們知道不定積分是在找微分前的函數，也就是反導函數或原函數：In this module, an indefinite integral means finding an anti-derivative.

∫ f(x) dx = F(x) + C

在上一個定積分模組中，我們知道定積分代表從 a 到 b 的累積量，可以用許多小矩形的有向貢獻逐步逼近：In the previous module, a definite integral represented accumulation from a to b, approximated by many signed rectangle contributions.

b∫a f(x) dx

接下來的「微積分基本定理」將會說明：The next module, the Fundamental Theorem of Calculus, will explain the following connection.

當 F'(x)=f(x) 時，微積分基本定理會說明為什麼 ∫ab f(x) dx = F(b) - F(a)。When F'(x)=f(x), the Fundamental Theorem of Calculus explains why the definite integral equals F(b)-F(a).

也就是為什麼辛苦算出來的「累積量」，竟然可以直接用「反導函數的端點差」來輕鬆計算！This explains why a difficult accumulation can be evaluated using the endpoint difference of an anti-derivative.

✨ 總結 ✨Summary

  1. 微分是已知 F(x)，求它的導函數 F'(x)，也就是 f(x)。
  2. 不定積分是已知 f(x) 回推 F(x)。
  3. 因為常數微分後是 0，所以反推的答案是一整族 F(x) + C。
  4. 不定積分的核心是尋找反導函數，它本身不是面積。
  5. 後續的微積分基本定理，會神奇地把不定積分（反導函數）和定積分（累積量）連接起來。
