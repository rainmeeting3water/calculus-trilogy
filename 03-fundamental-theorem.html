微積分基本定理：面積的變化率等於當下高度The Fundamental Theorem of Calculus: Connecting Accumulation and Anti-derivatives

從「累積面積」看懂微分與積分為什麼互相連接Use accumulation to see why differentiation and integration are connected.

三、用 4 個步驟理解證明Four Steps to the Argument

Step 1：定義面積函數Define the accumulation function

A(x) = ∫ax f(t) dt

意思是從起點 a 到位置 x 的累積面積。

Step 2：觀察新增面積Observe the newly added area

當位置從 x 變成 x+Δx，新增的面積是：

A(x+Δx) - A(x) = ∫xx+Δx f(t) dt

Step 3：把新增面積除以 Δx (找出平均高度)Divide the newly added area by Δx to find the average height

[A(x+Δx) - A(x)] / Δx

這個算式同時具有兩個重要意義：

  • 它是面積函數 A(x) 的平均變化率。
  • 它也是從 x 到 x+Δx 這段區間內，圖形的平均高度。

也就是說，同一個式子，一邊連到「導數」，一邊連到「面積平均高度」。

Step 4：讓 Δx 越來越小 (取極限)Let Δx approach 0

加上極限符號，這就是面積函數 A(x) 的導數定義：

A'(x) = limΔx→0 [A(x+Δx) - A(x)] / Δx

因為函數是連續的，x 附近的函數高度不會突然跳動，所以當區間縮得很小時，這一段的平均高度會越來越接近 f(x)。

(註：目前的互動圖展示的是 Δx>0，也就是從右側逼近。完整的導數還需要考慮 Δx<0 從左側逼近。因為這裡的 f 是連續函數，所以不論從右邊小區間或左邊小區間取平均高度，當區間寬度趨近 0 時，都會逼近同一個高度 f(x)。)

所以得到：A'(x) = f(x)

🎉 面積函數的瞬間變化率，就是當下的函數高度！The instantaneous rate of change of the accumulation function is the current height f(x).

四、從第一定理到第二定理From Part 1 to Part 2 of the Theorem

在高中階段，我們先討論 f 在區間上連續的情況。

第一部分我們已經知道面積函數的瞬間變化率就是當下高度：

A(x) = ∫ax f(t) dt

得到 A'(x) = f(x)

如果我們另外找到一個函數 F，它微分後會變回 f，也就是 F' = f，那麼 F 稱為 f 的反導函數，也常稱為原函數。

因為 A(x) 和 F(x) 的微分結果一樣：

直覺上，如果兩個函數在每一點的變化速度都一樣，那它們的圖形形狀相同，只可能上下平移，所以只差一個常數。

(A - F)' = A' - F' = f - f = 0

也就是說，A-F 的變化率一直是 0，所以 A-F 在這個區間上是一個常數。

所以：

A - F = C

也就是：

A(x) = F(x) + C

我們知道從起點到起點的累積面積一定是 0，也就是：

A(a) = ∫aa f(t) dt = 0

所以：

0 = F(a) + C

因此：

C = -F(a)

把 C 代回去，所以：

A(x) = F(x) - F(a)

最後代入 x = b：

∫ab f(t) dt = F(b) - F(a)

(註：若圖形在 x 軸下方，這裡的累積量會以帶符號面積理解。)

💡 第一定理說明：累積面積函數微分後，會回到原本的高度函數。Part 1: differentiating an accumulation function gives back the original height function.

💡 第二定理說明：只要找到一個微分後會變成 f 的函數 F，就可以用 F(b)-F(a) 快速算出從 a 到 b 的總累積量。Part 2: if F is an anti-derivative of f, then the definite integral is F(b)-F(a).

五、觀察例題Worked Examples

例題 1：拋物線下的面積Example 1: Area under a parabola

設定 f(x) = x² + 1，起點 a = 0

定義面積函數：A(x) = ∫0x (t² + 1) dt

經過計算（你可以先觀察圖形），真正的面積公式是：

A(x) = x³/3 + x

現在我們對面積函數微分：

A'(x) = 3x²/3 + 1 = x² + 1 = f(x)

💡 驗證成功： 先累積面積，再對面積函數微分，會回到原本的高度函數。
例題 2：直線下的面積Example 2: Area under a straight line

設定 f(x) = 2x + 1，起點 a = 0

定義面積函數：A(x) = ∫0x (2t + 1) dt

這個面積是一個梯形，算出來的結果是：

A(x) = x² + x

對面積函數微分：

A'(x) = 2x + 1 = f(x)

💡 再次強調： A'(x) = f(x) 的意思是，面積函數的變化率等於原本函數的高度。

✨ 總結 ✨Summary

微積分基本定理不是單純背公式，而是在說：

積分負責累積，微分負責看瞬間變化。
當我們累積的是曲線下面積時，面積增加的瞬間速度，就是當下的函數高度。

第一定理Part 1

d/dx ∫ax f(t)dt = f(x)

第二定理Part 2

∫ab f(t)dt = F(b) - F(a)，其中 F' = f

不同教材對第一定理、第二定理的命名可能略有不同；本模組把『累積面積微分回到原本函數』稱為第一部分，把『用反導函數端點差計算定積分』稱為第二部分。Different textbooks may label these parts differently; here we focus on the two connected ideas.
