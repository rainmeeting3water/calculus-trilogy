# 雙語數學詞彙表

本表整理「微積分三部曲」雙語版使用的核心中英術語。英文優先採用 IBDP Mathematics、國際高中數學教材與英美數學課堂常見用語，避免逐字直譯。

| 中文術語 | English Term | 說明 | 出現位置 | IBDP / International Usage Notes |
|---|---|---|---|---|
| 微積分 | calculus | 研究變化率與累積量的數學分支。 | 首頁、README | Standard international term. |
| 定積分 | definite integral | 表示一段區間上的累積量，可解讀為帶符號面積。 | 首頁、01 模組 | Commonly interpreted as accumulation over an interval or signed area. |
| 不定積分 | indefinite integral | 表示所有反導函數所形成的函數族。 | 首頁、02 模組 | Can be interpreted as a family of curves. |
| 微積分基本定理 | Fundamental Theorem of Calculus | 連接微分與積分的核心定理。 | 首頁、03 模組 | Standard title in IBDP and international calculus. |
| 微分 | differentiation | 從函數求導函數的運算。 | 02、03 模組 | IBDP uses differentiation for the process of finding a derivative. |
| 積分 | integration | 累積或尋找反導函數的運算。 | 三個模組 | IBDP uses integration for both definite and indefinite integral contexts. |
| 導函數 | derivative | 描述函數變化率的函數。 | 02、03 模組 | Standard IBDP term. |
| 反導函數／原函數 | anti-derivative / antiderivative | 微分後會得到原本函數的函數。 | 首頁、02、03 模組 | IBDP calculus content refers to anti-differentiation and anti-derivatives; avoid original function. |
| 反微分 | anti-differentiation | 由導函數反推反導函數的過程。 | 首頁、02 模組 | Prefer anti-differentiation over reverse differentiation. |
| 函數 | function | 對每個輸入給出對應輸出的規則。 | 三個模組 | Standard term. |
| 函數族 | family of functions / family of curves | 由參數 C 形成的一整族函數圖形。 | 首頁、02 模組 | For graphs of indefinite integrals, family of curves is natural. |
| 常數 C | constant C | 不定積分中代表任意垂直平移的常數。 | 首頁、02 模組 | The constant C shifts the graph vertically. |
| 斜率 | gradient | 表示直線或切線的傾斜程度。 | 02 模組 | IBDP and UK-style mathematics commonly use gradient; slope may be added in parentheses. |
| 變化率 | rate of change | 量隨自變數變化的快慢。 | 02、03 模組 | Standard international phrasing. |
| 瞬間變化率 | instantaneous rate of change | 在某一點的變化率，也就是導數值。 | 首頁、03 模組 | Common calculus classroom phrase. |
| 平均變化率 | average rate of change | 一段區間上的平均變化量除以寬度。 | 03 模組 | Used when interpreting difference quotients. |
| 面積 | area | 幾何區域大小。 | 01、03 模組 | Standard term. |
| 曲線下的面積 | area under the curve | 曲線與 x 軸之間的面積，常用於定積分直觀。 | 01、03 模組 | Standard classroom phrase. |
| 曲線與 x 軸所圍成的面積 | area of the region between the curve and the x-axis | 強調幾何區域本身。 | 01 模組 | Useful when distinguishing geometric area from signed area. |
| 面積函數 | area function / accumulation function | 以上限 x 為變數的累積函數。 | 首頁、01、03 模組 | For A(x)=∫[a to x]f(t)dt, accumulation function is often clearer. |
| 累積量 | accumulation | 一段區間上累加起來的總量。 | 首頁、01、03 模組 | Natural term for definite integrals in international teaching. |
| 累積函數 | accumulation function | 以變動上限描述累積量的函數。 | 03 模組 | Preferred for A(x)=∫[a to x]f(t)dt. |
| 新增面積 | newly added area | x 增加到 x+Δx 時增加的面積。 | 03 模組 | Natural phrase for explaining FTC visually. |
| 平均高度 | average height | 新增面積除以寬度 Δx。 | 首頁、03 模組 | Helps connect area and difference quotient. |
| 當下高度 | current height / instantaneous height | 在目前位置 x 的函數高度 f(x)。 | 03 模組 | Use current height f(x) when tied to the graph. |
| 小矩形 | small rectangle | 用來近似曲線下面積的小矩形。 | 首頁、01 模組 | Standard visual explanation for Riemann sums. |
| 矩形近似 | rectangle approximation | 用矩形估計累積量的方法。 | 首頁、01 模組 | Natural classroom phrasing. |
| 黎曼和 | Riemann sum | 小矩形貢獻的總和。 | 首頁、01 模組 | Standard term. |
| 取樣方式 | sampling method | 選取小矩形高度位置的方法。 | 01 模組 | Sampling method is natural in Riemann sum contexts. |
| 取樣點 | sample point | 每段用來取函數高度的點。 | 01 模組 | Standard phrase in Riemann sum definitions. |
| 左端點 | left endpoint | 每段左側端點。 | 01 模組 | Standard term. |
| 右端點 | right endpoint | 每段右側端點。 | 01 模組 | Standard term. |
| 中點 | midpoint | 每段中央的取樣點。 | 01 模組 | Standard term. |
| 帶符號面積 | signed area | x 軸上方為正、下方為負的面積解讀。 | 首頁、01 模組 | A definite integral can be interpreted as signed area. |
| 有向寬度 | directed width | 帶方向的 Δx，可能為正或負。 | 01 模組 | Useful when discussing reversed limits. |
| 有向貢獻 | signed contribution | 每個小矩形對總和的正負貢獻。 | 首頁、01 模組 | Signed contribution is natural when explaining signed area. |
| 滑桿 | slider | 調整參數的互動控制元件。 | 三個模組 | Standard UI term. |
| 測驗 | quiz | 檢查理解的小題目。 | 三個模組 | Standard classroom term. |
| 驗證 | verify | 檢查結果是否成立。 | 02、03 模組 | Use verify for checking by differentiation. |
| 估計 | estimate | 以近似方式取得數值。 | 01 模組 | Standard mathematical action verb. |
| 解釋／解讀 | explain / interpret | 說明原因或解讀圖像與數值意義。 | 三個模組 | Use interpret for meaning; use explain for reasoning. |
