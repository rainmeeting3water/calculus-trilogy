STEP 1｜先觀察函數圖形Observe the graph first

學生任務 Student Task

觀察函數圖形在 x 軸上方或下方的區域。

Observe where the graph is above or below the x-axis.

教師提問 Teacher Prompt

函數什麼時候在 x 軸上方？什麼時候在 x 軸下方？

When is the function above or below the x-axis?

觀察重點 Key Observation

定積分不只是幾何面積，也要注意帶符號面積。

A definite integral is not only about geometric area; it can also be interpreted as signed area.

STEP 2｜用小矩形近似累積量Approximate accumulation with small rectangles

學生任務 Student Task

調整 n，觀察小矩形如何覆蓋曲線下方的區域。

Adjust n and observe how the small rectangles cover the region under the curve.

教師提問 Teacher Prompt

每一個小矩形代表什麼？

What does each small rectangle represent?

觀察重點 Key Observation

每一個小矩形代表一小段帶符號貢獻。

Each small rectangle represents a signed contribution over a small interval.

STEP 3｜比較取樣方式Compare sampling methods

學生任務 Student Task

切換起點端點、終點端點與中點取樣方式。

Switch between the starting endpoint, ending endpoint, and midpoint sampling methods.

教師提問 Teacher Prompt

取樣方式會如何影響近似結果？

How does the sampling method affect the approximation?

觀察重點 Key Observation

不同取樣方式會造成不同估計，但 n 變大時會趨近相同累積量。

Different sampling methods can give different estimates, but as n becomes larger, they approach the same accumulation.

STEP 4｜讓 n 變大Let n become larger

學生任務 Student Task

增加 n，觀察矩形和曲線的貼合程度。

Increase n and observe how closely the rectangles fit the curve.

教師提問 Teacher Prompt

當矩形數量增加時會發生什麼事？

What happens as the number of rectangles increases?

觀察重點 Key Observation

這是黎曼和走向定積分的核心直觀。

This is the key intuition behind a Riemann sum approaching a definite integral.

STEP 5｜連到定積分符號Connect to the definite integral notation

學生任務 Student Task

用自己的話說明 ∫[a 到 b] f(x) dx 代表什麼。

Explain in your own words what ∫[a to b] f(x) dx represents.

教師提問 Teacher Prompt

黎曼和如何引導我們得到定積分？

How does the Riemann sum lead to the definite integral?

觀察重點 Key Observation

定積分表示一個區間上的累積量，也可以理解為帶符號面積。

A definite integral represents accumulation over an interval and can be interpreted as signed area.
