STEP 1｜定義累積函數Define the accumulation function

學生任務 Student Task

觀察 A(x)=∫[a 到 x] f(t) dt 的意義。

Observe the meaning of A(x)=∫[a to x] f(t) dt.

教師提問 Teacher Prompt

當 x 改變時，A(x) 累積了什麼？

What does A(x) accumulate as x changes?

觀察重點 Key Observation

A(x) 是一個累積函數。

A(x) is an accumulation function.

STEP 2｜觀察新增面積Observe the newly added area

學生任務 Student Task

觀察 x 增加 Δx 時，多出來的面積。

Observe the extra area added when x increases by Δx.

教師提問 Teacher Prompt

A(x+Δx)-A(x) 代表什麼？

What does A(x+Δx)-A(x) represent?

觀察重點 Key Observation

它代表短區間上的新增累積量。

It represents the newly added accumulation over a short interval.

STEP 3｜比較平均高度與當下高度Compare average height and current height

學生任務 Student Task

觀察 [A(x+Δx)-A(x)]/Δx 與 f(x) 的差距。

Compare [A(x+Δx)-A(x)]/Δx with f(x).

教師提問 Teacher Prompt

當 Δx 變小時會發生什麼事？

What happens when Δx becomes smaller?

觀察重點 Key Observation

平均高度會接近當下高度 f(x)。

The average height approaches the current height f(x).

STEP 4｜理解 A'(x)=f(x)Understand A'(x)=f(x)

學生任務 Student Task

用自己的話說明為什麼累積函數的瞬間變化率等於當下高度。

Explain in your own words why the instantaneous rate of change of the accumulation function equals the current height.

教師提問 Teacher Prompt

為什麼累積函數的導函數會等於被累積的函數？

Why is the derivative of the accumulation function equal to the function being accumulated?

觀察重點 Key Observation

累積函數的導函數就是被累積的函數。

The derivative of the accumulation function is the function being accumulated.

STEP 5｜連到 F(b)-F(a)Connect to F(b)-F(a)

學生任務 Student Task

理解如果 F'(x)=f(x)，則 ∫[a 到 b] f(x) dx = F(b)-F(a)。

Understand that if F'(x)=f(x), then ∫[a to b] f(x) dx = F(b)-F(a).

教師提問 Teacher Prompt

反導函數如何幫助我們計算定積分？

How does an anti-derivative help us evaluate a definite integral?

觀察重點 Key Observation

微積分基本定理連接了微分與積分。

The Fundamental Theorem of Calculus connects differentiation and integration.
