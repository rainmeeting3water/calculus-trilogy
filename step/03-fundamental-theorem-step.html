STEP 1｜定義累積函數Define the accumulation function

學生任務 Student Task

教師提問 Teacher Prompt

What does A(x) accumulate as x changes?

觀察重點 Key Observation

A(x) is an accumulation function.

STEP 2｜觀察新增面積Observe the newly added area

學生任務 Student Task

觀察 x 增加 Δx 時，多出來的面積。

教師提問 Teacher Prompt

What does A(x+Δx)-A(x) represent?

觀察重點 Key Observation

It represents the newly added accumulation over a short interval.

STEP 3｜比較平均高度與當下高度Compare average height and current height

學生任務 Student Task

觀察 [A(x+Δx)-A(x)]/Δx 與 f(x) 的差距。

教師提問 Teacher Prompt

What happens when Δx becomes smaller?

觀察重點 Key Observation

The average height approaches the current height f(x).

STEP 4｜理解 A'(x)=f(x)Understand A'(x)=f(x)

學生任務 Student Task

用自己的話說明為什麼面積函數的變化率等於當下高度。

教師提問 Teacher Prompt

Why is the derivative of the accumulation function equal to the function being accumulated?

觀察重點 Key Observation

The derivative of the accumulation function is the function being accumulated.
